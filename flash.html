CYD Remote Control

Flash the CYD Remote Control firmware to your ESP32 device using your browser.

Requirements:
  • Chrome, Edge, or Opera browser (must support Web Serial API)
  • ESP32 board (NodeMCU-32S or compatible)
  • USB cable for connection
Installation Steps:
  1. Connect your ESP32 board to your computer via USB
  2. Click the "Connect" button below
  3. Select your ESP32 device from the popup window
  4. Click "Install" and wait for the process to complete
  5. The device will reboot automatically when done
⚠️ Important: This will erase all existing data on the device.

Current Configuration:
WiFi SSID: Piste_001
UDP Target: 192.168.4.1:1234

To change these settings, you'll need to rebuild the firmware with updated values.
